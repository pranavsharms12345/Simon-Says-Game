# Simon-Says-Game
Simon Says is a memory-based game where the player must repeat a sequence of colors or actions shown by the system.  At the start, the game shows one color. With each level, a new color is added to the sequence. The player must remember and repeat the full sequence correctly.
📋 Features
1. Progressive Difficulty: Each level adds one more button to the sequence
2. Visual Feedback: Buttons flash with distinct colors and animations
3. Score Tracking: Real-time display of current score (number of completed rounds)
4. Game Over State: Clear feedback when sequence is incorrect with restart      option
5. Responsive Design: Adapts to different screen sizes and devices
6. Keyboard Controls: Press any key to start/restart the game
7. Sound Effects: Optional  audio feedback for button presses and actions
8. High Score Persistence: Tracks and displays best scores
 🎮 How to Play
1. Start the Game: Press any key on your keyboard to begin
2. Watch the Sequence: Simon will display a sequence of colored button flashes
3. Remember the Pattern: Pay close attention to the order and colors shown
4. Repeat the Sequence: Click the buttons in the exact same order as Simon displayed them
5. Advance Levels: Successfully completing a sequence progresses you to the next level
6. Game Over: Making a mistake ends the game and displays your final score.
🔧 Technical Details
Technologies: HTML5, CSS3, JavaScript
Framework: Vanilla JavaScript (no dependencies).
